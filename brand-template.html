07  ·  Theme Rules

Design Principles

Dark-First

The base is always --black (#050c08). Surfaces layer up with progressively lighter green-blacks to create depth without light mode.

Teal = Action

Teal is used for interactive elements, labels, borders, and hover states. It signals "something you can do" or "something important."

Gold = Emphasis

Gold is the secondary accent. Used sparingly in gradients, badges, callout borders, and stat numbers. Never for buttons or links.

Gradient Text

Italicized words in section titles use a teal-to-gold gradient fill via background-clip. This is the signature accent treatment.

No Border Radius

All cards, buttons, and containers use sharp corners. The only exception is circular avatars (border-radius: 50%).

1.5px Grid Gaps

Card grids use 1.5px gaps with a colored background behind them, creating thin visible separators instead of traditional spacing.
